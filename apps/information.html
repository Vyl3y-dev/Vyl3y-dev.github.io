Welcome to VeeOS

VYL3Y-DEV.GITHUB.IO

This site is designed as a playful, desktop-style environment — a fusion of creativity, interactivity, and development. Each window acts as its own “app,” loaded dynamically just like a mini operating system running in your browser.

👩‍💻 About Me

Hi! I’m Vyley — an amateur developer, designer, and digital tinkerer who loves building interactive, functional systems with personality. This website is both my portfolio and sandbox, showing off the projects I’ve built, the tools I use, and the ideas I’m experimenting with.

🧩 About VeeOS

  • Built entirely with HTML, CSS, and Vanilla JavaScript
  • Hosted on GitHub Pages
  • Styled to resemble a retro-futuristic desktop interface
  • Dynamically loads “apps” into draggable, resizable windows
  • Local testing powered by python -m http.server or npx serve
Each app lives in the /apps/ directory and loads via fetch() in script.js.

🔧 How it Works

When you open an app:

  1. script.js fetches its .html file from /apps/.
  2. Linked styles (.css) and scripts (.js) are dynamically loaded.
  3. Each app runs in its own sandboxed window.
Windows can be dragged, minimized, and later made resizable.

🎨 Design Philosophy

“Form follows function — but fun follows form.”

I believe tools should not only work well, but also feel delightful to use. That’s why this site merges aesthetics from:

  • Classic operating systems
  • Early web apps and pixel UIs
  • Modern flexibility and motion

📱 Tech Stack

  • Frontend: HTML, CSS, Vanilla JS
  • Backend (local dev): Python simple server
  • Version Control: Git + GitHub
  • Deployment: GitHub Pages
  • Planned Features: Local storage for app state, draggable/resizable windows, custom boot screen.
